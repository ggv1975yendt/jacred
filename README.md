# TorrentSeek 🔍

Веб-приложение на Go для поиска торрентов на rutor.info.  
Аналог Jackett — единый интерфейс для торрент-трекеров.

## Возможности

- 🔍 Поиск по названию на **rutor.info**
- 📋 Отображение результатов в таблице: название, размер, сиды, пиры, дата
- 🧲 Прямые **магнет-ссылки** для открытия в торрент-клиенте
- 📋 Кнопка **копирования** магнет-ссылки
- 🌐 JSON API (`/api/search?q=...`)
- 🎨 Тёмный интерфейс в стиле терминала

## Установка и запуск

### Требования
- Go 1.21+

### Быстрый старт

```bash
# Скачать зависимости и запустить
make run

# Или вручную:
go mod tidy
go run .
```

Откройте браузер: **http://localhost:8080**

### Сборка бинарника

```bash
make build
./torrent-search
```

### Docker

```bash
docker build -t torrent-search .
docker run -p 8080:8080 torrent-search
```

### Переменные окружения

| Переменная | По умолчанию | Описание |
|-----------|-------------|---------|
| `PORT`    | `8080`      | Порт сервера |

## API

### Поиск (HTML)
```
GET /search?q=терминатор
```

### Поиск (JSON)
```
GET /api/search?q=терминатор
```

**Пример ответа:**
```json
{
  "query": "терминатор",
  "count": 15,
  "results": [
    {
      "name": "Терминатор / The Terminator (1984)",
      "magnet": "magnet:?xt=urn:btih:...",
      "size": "2.1 GB",
      "seeds": "42",
      "peers": "7",
      "date": "12.03.2024",
      "info_hash": "ABC123..."
    }
  ]
}
```

## Расширение

Для добавления новых трекеров создайте функцию по образцу `searchRutor()` в `main.go` и вызывайте её параллельно через goroutines.
